## Getting Started

1. Browse the templates on [SiteReadyPro](https://grchetan.github.io/sitereadypro/)
2. Click the Download button on any template to go to its GitHub repo
3. Clone or download the ZIP from the repo:
   ```bash
   git clone https://github.com/grchetan/<template-name>.git
   ```
4. Open `index.html` in your browser or deploy it anywhere!
